# 모바일 프로그래밍 - try-Catch

---

## 앱 목적
사용자의 **프롬프트(prompt) 작성 실력 향상**을 위해 만들어진 앱입니다.

## 주요 기능
- **1. 사용자 정보 등록**
  - 닉네임 입력 후 중복 체크
  - 
- **2. 레벨 테스트**
  - 처음 3문항 정도로 사용자의 실력 파악
  - 
- **3. 문제 풀이**
  - 문제 제공
  - 답변 작성 후 제출
  - 
- **4. 피드백 제공**
  - 답변에 대한 평가 및 간단 피드백 확인
  - 
- **5. 기록 조회**
  - 최근 작성한 답변 30개 확인 가능
  - 점수 및 풀이 횟수 순위 확인
